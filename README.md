# Build Modernization and Flaky Tests in Apache Kafka

This talk was presented at [Community Over Code NA 2025](https://communityovercode.org/) and as a 
[Gradle Webinar](https://gradle.com/events/modernizing-apache-kafka-ci-builds-develocity-github-actions-03-26/). 
It covers the build improvements we made to Apache Kafka in late 2024 and how we have managed flaky tests since then.

Some of this content is also discussed on my Subtack, [Building Apache Kafka](https://mumrah.substack.com).

If you have questions or feedback, feel free to open an Issue here or email me.
